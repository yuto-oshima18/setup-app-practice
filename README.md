# setup-app-practice

## 概要
COACHTECH Laravel基礎 「9-1-7: 環境構築ハンズオン」にて作成

## 使用技術
- PHP 8.5.8
- Laravel 10.50.2
- Laravel Sail8.5（Docker Compose）
- MySQL8.4.1, phpMyAdmin5.2.3

## 学んだこと
- Laravel Sailのプロジェクト作成手順
- PHP・MySQL・phpMyAdminのコンテナ作成手順
- sailから作成したコンテナの動作確認手順

## 動作確認
- ①sailコマンドによりバックグラウンドで起動(sail up -d)
- ②sailコマンドによるコンテナ状態確認(sail ps)
- ③Laravelアプリケーション(http://localhost)、phpMyAdmin(http://localhost:8080)にアクセス
